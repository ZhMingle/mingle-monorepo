🎯 关键信息

window.innerHeight: 浏览器窗口的可视区域高度（不包括浏览器UI）

document.documentElement.clientHeight: HTML元素的可视区域高度

screen.height: 设备屏幕的总高度

getBoundingClientRect().top: 元素相对于视口顶部的距离

💡 使用建议

在你的 React 应用中，使用 getBoundingClientRect().top 可以准确获取元素距离视口顶部的距离，这样就能动态计算出状态栏的高度了。
